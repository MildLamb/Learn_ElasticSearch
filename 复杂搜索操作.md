# ES复杂的查询操作
- 查询的参数体以json字符串传递
```Java
GET mildlamb/_search
{
  //查询参数以json字符串传递
  "query": {  
    "match": {
      "name": "kindred"
    }
  },
  //指定显示的字段,结果的过滤
  "_source": ["name","desc"]
}
```
- 查询结果
  - hit:(后面会用java操作这个对象)

    - 查询的结果总数
